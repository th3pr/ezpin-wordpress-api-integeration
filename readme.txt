You can now use electronic cards directly through WooCommerce.

== Description ==

EZpin API plugin allows WooCommerce store owners to integrate electronic cards directly into their online store. This plugin interacts with the EZpin API to manage electronic card products, handle orders, and keep product data synchronized.

= Features =
* Fetch and display electronic card products from EZpin API.
* Manage electronic card orders seamlessly within WooCommerce.
* Automatic synchronization of products with the EZpin API.
* Secure authentication and API access token management.

= Installation =
1. Upload the plugin files to the `/wp-content/plugins/ezpin-api` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin settings by navigating to the 'EZpin Settings' in the WordPress admin menu.
4. Set your EZpin API client ID and secret key.
5. Synchronize products by using the provided synchronization tools.

= Usage =
* Ensure you have WooCommerce installed and activated.
* Configure your API credentials in the EZpin Settings.
* Use the synchronization tools to fetch and update product data from the EZpin API.
* Manage orders containing EZpin products directly through WooCommerce.

== Frequently Asked Questions ==

= Does this plugin require any additional plugins? =
Yes, this plugin requires WooCommerce to be installed and activated.

= How do I obtain my EZpin API credentials? =
You can obtain your API credentials by signing up for an account on the EZpin platform and generating an API client ID and secret key.

= How often are products synchronized with the EZpin API? =
You can configure the synchronization interval in the plugin settings. By default, the plugin synchronizes products every hour.

= What should I do if the plugin encounters an error? =
Ensure your API credentials are correct and that your server can communicate with the EZpin API. Check the plugin documentation for troubleshooting tips.
